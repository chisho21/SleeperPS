# Get-SleeperNFLState

## SYNOPSIS
Retrieves the current state of the NFL season.

## SYNTAX

```
Get-SleeperNFLState
```

## DESCRIPTION
The Get-SleeperNFLState cmdlet retrieves the current state of the NFL season, including the current week and phase (regular season, playoffs, etc.).

### EXAMPLE 1
```

```
